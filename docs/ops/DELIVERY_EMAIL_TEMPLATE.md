## Fill-in guide by tier

| Tier | Delivered bullets must include |
|------|-------------------------------|
| Friction Check | Exact URL reviewed · Loom/video link · Written takeaways |
| First Fix | Exact change made (match scope email) · Before/after if live · How to verify |
| Mini Build | Each milestone item · Test steps · Access notes if applicable |

**Acceptance deadline:** Delivery date + 7 calendar days.

## After send (mandatory)

| Step | Action |
|------|--------|
| 1 | `Deliverables/` — Loom link, screenshots, files |
| 2 | `Acceptance/delivery-[YYYY-MM-DD].txt` — paste full email |
| 3 | Sheet: **Delivered at** → today |
| 4 | Sheet: **Sent to email** → address used (prefer Stripe payer email) |
| 5 | Sheet: **Acceptance deadline** → today + 7 days |
| 6 | Sheet: **Lifecycle** → `Acceptance Window` |
